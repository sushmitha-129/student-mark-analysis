# Student Marks Analysis

## Objective
Analyze student performance using Python and basic statistical methods.

## Dataset
CSV file containing student marks and attendance with missing values.

## Steps Performed
- Data loading and inspection
- Missing value handling
- Feature engineering
- Statistical analysis
- Performance evaluation

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Outcome
Extracted insights on student performance and pass probability.
